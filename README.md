# Relatorio Automatico Python



um script que lê uma planilha de dados e envia um resumo formatado por e-mail, sem você precisar fazer nada manualmente.


O quê o projeto faz:
Lê uma planilha CSV com dados de vendas: Você vai criar um arquivo simples com produto, quantidade e valor
Calcula totais e o produto mais vendido: O script faz os cálculos automaticamente
Envia o resumo por e-mail automaticamente: Com assunto, corpo formatado e os dados calculados


## Como usar

Execute o arquivo relatorio-automatico-python e siga as instruções.
